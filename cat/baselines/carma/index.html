CARMA Baseline Solutions

Applying Baseline

  • Baseline correction:
    • Do a miriad cvs update to your source tree. This way, you will have the latest baseline solutions in $MIRCAT/baselines/carma/antpos.YYMMDD file. 
               cd $MIR ; cvs -nq update ; cvs update
  • Alternately,
    • Find the date your data were taken, and download a baseline solution from the list below.
    • Apply the solution "antpos.YYMMDD" to the data "xxx.mir", and obtain the corrected data "yyy.mir". 
               uvedit vis=xxx.mir apfile=antpos.YYMMDD out=yyy.mir
    • It's safe to apply baseline correction if you don't know what solution has been applied to your data.

  • Axis offset correction:
    • Early tracks (before Jan 5, 2007) need axis offset correction (don't forget @). 
               axcor vis=yyy.miriad axoff=@axoff.xxx out=zzz.miriad
